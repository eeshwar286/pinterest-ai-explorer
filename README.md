# ✨ Pinterest AI Explorer

A lightweight floating AI assistant that makes Pinterest search way more creative.

It watches what you're searching and suggests fresh, smarter queries in real time.

---

## 🚀 Features

### 🧠 Smart Suggestions
Get AI-powered search ideas based on your current and recent Pinterest queries.

### 🔄 SPA-Aware
Works smoothly with Pinterest’s single-page app — detects URL changes and updates automatically.

### 🫧 Draggable Bubble
A floating ✨ button you can move anywhere on your screen.

### ⚡ One-Click Apply
Click a suggestion → it copies + auto-fills into Pinterest search instantly.

### 🧩 Smart Memory
Remembers your last **3 searches** to improve suggestion quality.

---

## 📦 Installation

Follow these steps to install the extension locally:

1. **Download the ZIP**
2. **Unzip the file**
3. Open Chrome and go to: chrome://extensions
4. Enable **Developer Mode** (top-right toggle)
5. Click **Load unpacked**
6. Select the unzipped folder
7. Click the extension icon and paste your **Gemini API Key**

---

## 🧪 How to Use

1. Go to 👉 https://www.pinterest.com  
2. Run any search  
3. A ✨ floating bubble appears (bottom-right)
4. Click it to open suggestions
5. Tap any suggestion to instantly use it

---

## 💡 Example

Search: minimalist bedroom
